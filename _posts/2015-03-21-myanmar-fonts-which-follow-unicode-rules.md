**Padauk**

Font family &#8211; padauk

Version &#8211; 2.6.1

Known Issue &#8211; In a mixture of English and Myanmar texts in Firefox in Windows 7, English texts may lose kerning and combine together rendering unreadable. Update:11June2011# Padauk font English glyphs will have same bug on Firefox 4+ on any version of Windows. Update:13 July 2016# It&#8217;s not font issue, rather Firefox turning the Graphite features off (which Graphite is a rendering engine installed in the font itself). [Here is how to fix it](http://scripts.sil.org/cms/scripts/page.php?site_id=projects&item_id=graphite_firefox) (You can find the solution at the end of the page).
